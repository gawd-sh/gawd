# Trademark Policy

The **Gawd™** name, the **trident mark**, and the **Founder's Sigil** identify this project. The code is MIT-licensed; these marks are not. This policy is deliberately short and permissive — it exists to prevent impersonation, not to bother fans.

## You're welcome to (no permission needed)

- Refer to the project as "Gawd" in articles, posts, talks, videos, and documentation
- Take and publish screenshots of Gawd, including the marks as they appear in the product
- Say your software "works with Gawd," "is built on Gawd," or "is a fork of Gawd" (truthfully)
- Link to this repository or gawd.sh using the project name

## Please don't (without written permission)

- Use "Gawd" or the marks as the name or logo of your own product, fork, service, or company
- Publish modified versions of the trident or Sigil
- Use the marks in a way that implies your work is official or endorsed by this project
- Sell merchandise carrying the marks

## Forks

Forking the code is what the MIT license is for — go forth. A distributed fork should pick its own name and logo; "based on Gawd" in your README is the right way to credit it.
