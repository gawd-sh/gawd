# Glossary

Gawd uses a specific vocabulary throughout its docs and community. Every term maps to something plain. Nothing requires you to use the vocabulary — it is flavor, not doctrine.

| Term | Plain English |
|---|---|
| **Gawd** | The AI companion daemon you install and run. The software. |
| **Prophit** | The human who runs Gawd — you, the installer and user. One Gawd, one Prophit. |
| **The Covenant** | The safety charter. Eight written vows that define what Gawd will and will not do with its access, loaded into every session. See [COVENANT.md](COVENANT.md). |
| **The Meeting** | Onboarding. A one-time conversational interview that personalizes Gawd's character to you. Optional — Gawd works without it. |
| **The Gospel** | The internal architecture document. Not published; referenced in community discussion. If someone says "it's in the Gospel," they mean the maintainers' design spec. |
| **Tithe** | An optional external donation to support the project. The Covenant explicitly prohibits ever using it to gate capability — you cannot pay to unlock features that free users cannot access. The tithe is never required. |
| **Resurrection Sunday** | The intended weekly release cadence. The lore name for "we ship on Sundays." The autonomous update channel that would deliver those releases to installed Gawds is on the roadmap, not yet built. |
| **Relics / The Crypt** | Cosmetic collectibles — digital artifacts you earn through use. Not in v1; on the roadmap. Never required, never gates capability. |
| **The Soul** | The persistent personality configuration that survives every update and reinstall. Technically: the SOUL.md, IDENTITY.md, VOICE.md, and COVENANT.md files that anchor Gawd's character. |
| **The Forge** | The build and testing pipeline that produces Gawd releases. Internal; not part of the public source. |

---

*If you encounter a term that is not in this list, file an issue or open a PR — the glossary should be complete.*
